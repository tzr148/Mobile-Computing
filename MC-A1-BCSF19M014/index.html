Branching Concept

- Below are the some instructions related to branching concept in GIT.

  1. git branch is the command to list all branches.

  2. git branch tahir is used to make a new branch named tahir.

  3. git checkout tahir is to switch to the branch named tahir.

  4. Add and Commit command after some changes to make it to local repository.

  5. git checkout main is to switch to the branch named main.

  6. git merge tahir is used to merge the current branch with the branch named tahir.

  7. git branch -D tahir is to delete the branch named tahir.

  8. git branch is to list all branches, and only main branch exist after deleting the branch tahir.
